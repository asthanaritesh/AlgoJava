<1> Mediator design pattern is one of the behavioral design pattern, so it deals with the behaviors of objects. 
<2> Mediator design pattern is used to provide a centralized communication medium between different objects in a system.
<3> Mediator promotes loose coupling by keeping objects from referring to each other explicitly, 
and it lets you vary their interaction independently.
<4> Air traffic controller is a great example of mediator pattern where the airport control room works as a mediator for communication between different flights.
<5> Mediator works as a router between objects and it can have it’s own logic to provide way of communication.

example:
It is a chat application where users can do group chat. Every user will be identified by it’s name and they can send and receive messages. 
The message sent by any user should be received by all the other users in the group.(except himself)
